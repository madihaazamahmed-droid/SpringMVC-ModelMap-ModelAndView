# Spring MVC - ModelMap & ModelAndView

A Spring MVC project demonstrating how to pass data from a controller to JSP views using `ModelMap` and `ModelAndView` with Java-based configuration.

## Features

- Spring MVC Java Configuration
- DispatcherServlet Configuration
- Component Scanning
- ModelMap Example
- ModelAndView Example
- JSP View Rendering

## Technologies Used

- Java
- Spring MVC
- Maven
- JSP
- Apache Tomcat
- Eclipse IDE
